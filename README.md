A full-stack job portal web application built using the MERN stack (MongoDB, Express, React, and Node.js). The platform connects job seekers and recruiters through a modern, responsive interface.

## Features

### Job Seekers
- Search and filter job listings
- Apply for jobs online
- Create and manage user profiles
- Upload and manage resumes
- Secure authentication using Clerk

### Recruiters
- Recruiter dashboard
- Post and manage job listings
- View, accept, or reject applications
- Access applicant resumes

## Tech Stack
- Frontend: React, JavaScript, HTML, CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: Clerk
- Monitoring & Error Tracking: Sentry
